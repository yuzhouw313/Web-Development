Variables and Objects

  • Use let or const to define variables.
  • Arrays use [ ] brackets.
  • Objects are key-value containers and use { } braces.
  • Try using the window object to pop up an alert.
  • Watch out for type conversions! Use === instead of ==.

Modifying the DOM

  • querySelector() function to get a single element
  • querySelectorAll() function to get an array of matching elements
  • Use a for...of loop to iterate over an array.
  • Be sure to wait for the DOMContentLoaded event on the document object.

Events

  • Use addEventListener() to handle a specific event on a specific DOM element.
  • Experiment with monitorEvents() in the Console to see what's possible.

JavaScript Syntax and Cheat Sheet

  • This week, the images subfolder contains images of dice.
  • Use let or const to define variables.
  • Define page-wide functions with the function keyword.
  • You can also define functions by assigning them to variable names, like this: 
    
  let f = function(arg1, arg2) {
  // code goes here
  }
  • Given an element elem, you can attach a "click" event handler with: elem.addEventListener("click", myFunctionName)
  • Your event handler functions should accept one argument, which is the event object: 
    
    let myHandler = function(event_obj) { ... }
  • To prevent the browser from also responding to a click event, you can call event_obj.preventDefault();
  • When inside an event handler function, you can refer to the original element by using event.target.
